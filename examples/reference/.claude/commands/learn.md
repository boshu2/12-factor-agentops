# /learn - Learning Phase

**Purpose:** Extract patterns and improvements after implementation

**When to use:** After implementation complete, before moving to next project

**What it does:** Invokes learn-agent to extract reusable patterns

---

## Usage

```bash
/learn
```

**Agent will:**
1. Review completed work
2. Extract reusable patterns
3. Identify improvements
4. Document learnings
5. Share knowledge for future work

**Output:** Learning document + pattern library updates

---

**Demonstrates:** Factor IX (Pattern Extraction), Factor X (Continuous Improvement)
